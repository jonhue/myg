# Animate

## Usage

### Stylesheets

**Sass**

```sass
@import "myg-animate/animate"

// Include classes (optional)
+myg__classes--animation
```

### Javascript

**ES6**

```js
import MygAnimate from 'myg-animate';
```

or

```js
import * as myg from 'myg-src/myg';
const MygAnimate = myg.MygAnimate;
```
